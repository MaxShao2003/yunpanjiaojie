# 《云盘交接》

> 一款网页端沉浸式悬疑解谜游戏

## 游戏简介

你是一名刚入职「鼎铭文化传媒」的新员工，职位是视频剪辑师。

今天是你的第一天。HR 告诉你，前任员工赵宇已经离职，你需要在**今晚 24:00 前**完成他留下的工作交接——登录公司云盘，阅读并确认所有交接资料。

听起来只是一份普通的入职任务。

但当你打开云盘，翻阅那些看似平常的文件时，你开始注意到一些……不太对劲的地方。

考勤表上反复出现的异常时间。被涂黑的员工手册条款。项目文件夹里诡异的报销记录。还有，赵宇——他真的只是"离职"了吗？

**在这家公司，有些事情不该被知道。**

**但你已经看到了。**

## 游戏特色

🔍 **沉浸式办公系统模拟**  
高度还原企业 OA 云盘界面，让你仿佛真的在翻阅一个真实员工的工作资料。

🧩 **多层嵌套解谜**  
从公开文件到加密文件夹，从表面信息到隐藏线索——每一份文档都可能藏着通往真相的钥匙。

👁️ **细节即恐怖**  
没有跳吓，没有血腥画面。恐惧来自于那些"不该存在"的细节，和你逐渐拼凑出的可怕真相。

🎭 **多结局分支**  
你的选择将决定故事的走向。是选择视而不见，还是追查到底？

## 如何开始

### 在线游玩
直接访问游戏网址，点击「开始交接」即可进入。

### 本地游玩
1. 下载游戏文件
2. 用浏览器打开 `index.html`
3. 开始你的第一天

### 推荐环境
- 使用电脑浏览器游玩（Chrome / Firefox / Edge / Safari）
- 手机竖屏也可游玩，但电脑体验更佳
- 建议在安静、光线较暗的环境下游玩

## 玩法提示

> 💡 以下提示**不含剧透**，仅帮助你更好地体验游戏。

### 基本玩法
- **仔细阅读每一份文件**——文档中的任何信息都可能是线索
- **记住关键信息**——人名、工号、日期、时间……这些细节很重要
- **多处交叉验证**——不同文件之间的信息可能相互关联
- **尝试交互**——有些地方可以点击、输入，或者……连续点击

### 遇到密码锁？
游戏中有些文件夹需要密码才能访问。密码的线索一定藏在你已经能看到的资料里。

仔细回想：
- 有没有什么反复出现的词语或短语？
- 有没有什么明显被强调的信息？
- 有没有什么可以组合的元素？

### 卡关了怎么办
1. 重新翻阅已解锁的所有文件
2. 注意那些看似"无关紧要"的细节
3. 问问自己：这份文件想告诉我什么？

## 注意事项

⚠️ **本游戏包含轻度恐怖元素**  
虽然没有惊吓画面，但氛围和内容可能令人感到不安。请根据自身情况决定是否游玩。

🔊 **建议开启声音**  
部分页面包含音频内容，会增强游戏体验。

💾 **游戏会自动保存进度**  
你可以随时关闭浏览器，下次打开时会从上次的进度继续。  
如需重新开始，请清除浏览器的本地存储数据。

## 常见问题

**Q: 游戏大概需要多长时间通关？**  
A: 首次游玩约需 30-60 分钟，取决于你的探索速度和解谜能力。

**Q: 可以多人一起玩吗？**  
A: 当然可以！和朋友一起讨论线索、推理剧情是很棒的体验。

**Q: 我找不到某个线索怎么办？**  
A: 游戏设计上确保所有谜题都有明确的线索指向。如果卡关，建议休息一下再回来，往往会有新发现。
