### Logo 五连击机制

- 操作：在任何页面左上角 Logo 连续点击 5 次（3秒内）
- 解锁成功：跳转 P27（后台登录页）
- 解锁失败：跳转 P03（404错误页）

### localStorage 状态记录
