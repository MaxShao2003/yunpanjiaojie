【核心优化】

布局适配:
  桌面版  →  二列布局 (侧栏 + 内容)
  平板版  →  一列布局 (水平导航条)
  手机版  →  一列布局 (水平导航条，可滑动)

导航菜单:
  PC端    →  固定左侧竖向导航栏
  平板    →  顶部水平导航条，可滑动
  手机    →  顶部水平导航条，紧凑显示，可滑动

字体大小分级:
  标题    → 28px (桌面) / 22px (平板) / 20px (手机)
  正文    → 14px (桌面) / 13px (平板) / 12px (手机)
  说明    → 12px (桌面) / 11px (平板) / 10px (手机)

触屏优化:
  → 所有交互元素 ≥ 44px × 44px
  → 链接和按钮内边距充足
  → 悬停效果检测 (避免触屏卡顿)
  → 原生滚动体验 (-webkit-overflow-scrolling)

特殊组件:
  • 表格  → 自动启用水平滚动，数据完整
  • 图片  → 自适应宽度，竖屏下最大高度400px
  • 音频  → 全宽播放器，触屏控制
  • 文档  → 自适应容器，超大文本可读

【维护和扩展】

新页面添加:
  1. 确保包含 viewport meta 标签
  2. 链接 responsive.css 和 mobile.css
  3. 使用现有 CSS 变量和类名
  4. 可选: 添加页面特定的 media query

修改现有样式:
  1. 优先修改 base 样式（非响应式）
  2. 响应式调整在对应 media query 块中
  3. 保持 CSS 变量的一致性
  4. 避免硬编码像素值
