## 第一部分：零基础完整攻略

### 🎮 游戏概述

你是一名新员工，收到任务在24小时内完成前同事赵宇的离职资料交接。但随着逐页阅读，你会发现公司并不是一个真的媒体公司...

**时间限制**：游戏全程有"今晚24:00前"的时间压力，但实际不会强制触发，只增强紧迫感。

**核心机制**：
- 探索不同文件 → 收集线索
- 解开两个密码锁 → 获得关键信息
- 按特定顺序操作 → 解锁隐藏后台
- 做出最终选择 → 触发三种结局之一

## 附录：常见问题与解合案

### ❓ Q&A

**Q：登录凭证是什么？**
A：邮件中提供的临时账号，必须使用正确凭证才能登录。账号：`zy_transfer@dingming.local`，密码：`DMhandover0314`。使用错误的账号或密码会被拒绝登录。

**Q：P00首页的访问历史是什么？**
A：系统会自动记录你访问过的所有content页面（最多显示最近12条）。每访问一个新页面，它就会被自动添加到历史中。点击历史中的条目会以新标签页打开该页。

**Q：进度卡片显示的 0/42 是什么意思？**
A：0/42 表示你已经访问了 0 个content页面，总共有42个。随着你访问更多页面，这个数字会自动增加。

**Q：为什么顶部进度条显示的百分比和访问个数不一样？**
A：顶部的进度百分比（在login.html上）是基于**关键节点**计算的（12% → 28% → 43% → 51% → 63% → 78% → 87% → 100%），而 P00 首页显示的 0/42 是基于**实际访问页数**计算的。两套系统各有用途。

**Q：我找不到微信截图中的关键台词？**
A：查看P16里赵宇和同事的对话，"这破公司我真是不想干了"这句是关键。

**Q：我不记得赵宇的工号了？**
A：P07考勤表和P09通讯录都有 DM-8975。

**Q：密码一直输错怎么办？**
A：P15和P19都有多次尝试机会，别担心。如果还是不对，可以查看本攻略的"第三阶段"。

**Q：我怎么进不去后台？**
A：确保你已经：
- 读过P23日记3，学到"Logo五连击"的方法
- 获得 assetsUnlocked=true 状态（解开/assets文件夹）
- 在前台任意页面快速点击Logo 5次

**Q：P35倒计时结束了没有自动跳转？**
A：这是目前的BUG。你可以手动点击任意按钮完成游戏。

**Q：我想玩纯后台路线，跳过前面？**
A：不建议，因为：
- 会失去"逐步发现真相"的恐怖感
- 某些关键信息散落在前面页面
但如果想快速通关，看P23→P26→Logo五连击→后台 是最短路径。

**Q：三个结局有什么区别？**
A：
- **坏结局**：你被完整吸收，没有逃脱
- **普通结局**：你逃了但永远不会完全摆脱
- **真结局**：你摧毁了一部分，但更大的系统仍然存在

每个结局都有其独特的哲学含义。
