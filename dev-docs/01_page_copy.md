# 《交接云盘》40页逐页文案版

这份文档按“能直接写成页面”的标准整理。每页都包含页面定位、建议标题、可直接使用的中文文案、交互提示、线索落点与跳转说明。你可以把它当成游戏文案总表，也可以一页一页拆给 Cursor。

## 全局文案与气质统一

- 前台系统关键词：`交接`、`验收`、`同步`、`留存`、`状态稳定`、`夜间巡检`
- 后台系统关键词：`饲养舱`、`收割`、`材料`、`转化`、`清理程序`
- 反复出现的时间锚点：`03:14`、`22:00后`、`今晚24:00前`
- 情绪节奏：前10页尽量正常，11-18页出现裂缝，19-26页建立求生逻辑，27页后彻底翻面
